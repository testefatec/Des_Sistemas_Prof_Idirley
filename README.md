# Projeto: FATEC — Segurança da Informação (Disciplina: Desenvolvimento de Sistemas)

Este repositório contém um projeto didático para aulas práticas de Git/GitHub, CI/CD e deploy na AWS.

- Página estática: `index.html` (exemplo com logo da FATEC)
- Workflows GitHub Actions: CodeQL, secret-scan (gitleaks) e deploy para S3
- Dependabot configurado para checar dependências regularmente
- Workflow que notifica o professor por e-mail quando um PR é criado/atualizado

Leia a documentação completa em `docs/guide.md` para instruções passo-a-passo (fork, Git, ambientes, AWS, secrets, política IAM e roteiro de aulas).
